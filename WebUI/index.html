DevOps Genie

A free, responsive, one page Bootstrap themed DevOps framework using Angular2 and Azure deployed REST API's.
Created by Cloud Computing Associates

About DevOps Genie Project

DevOps Genie is an open source framework licensed under the MIT License The brainchild of Marty Mazurik, its an easy scaffold to begin managing your cloud with code.

This framework features bootstrap for NG2 (Angular2). I coded using VS 2015, stores in GitHub, UI deploys to Ubuntu using Jenkins for Continous Integration. REST APIs live on Azure, and call Azure Resource Manager API using OAuth2.
